# AIction 隐私政策

**最后更新：** 2026年6月4日

## 简介

本隐私政策描述了 AIction（"我们"、"我们的"或"本扩展"）在您使用我们的 Chrome 浏览器扩展时，如何收集、使用和保护您的信息。

## 我们收集的信息

### 1. 用户配置数据

**AI 服务配置：**
- API 基础 URL
- API 密钥
- 模型选择
- 模型参数（温度、最大 token 数等）

**用户偏好：**
- 主题设置（浅色/深色/自动）
- 语言偏好
- 自定义动作模板
- 自定义图标

**存储位置：** 所有配置数据使用 `chrome.storage.sync` 存储在您的浏览器本地，如果您登录了 Google 账号，配置会在您的 Chrome 实例之间同步。

### 2. 匿名使用分析

**我们收集的内容：**
- 扩展安装和更新事件
- API 连接测试结果（成功/失败、延迟）
- 匿名使用标识符（随机生成的 UUID）

**我们不收集的内容：**
- 您选中的文本内容
- 您与 AI 的对话
- 您的 API 密钥或凭据
- 个人身份信息
- 浏览历史
- 页面内容或 URL

**分析服务提供商：** 我们使用 PostHog 进行匿名使用分析。您可以随时在扩展设置中禁用分析功能。

### 3. 文本处理

当您选中文本并使用 AI 功能时：
- 选中的文本会发送到您配置的 AI 服务提供商
- 此处理通过您自己的 API 凭据进行
- 我们不会将您的文本存储或传输到我们的服务器
- 对话仅存储在内存中，关闭页面后即被清除

## 我们如何使用信息

### 配置数据
- 提供和维护扩展功能
- 在设备间同步您的设置（通过 Chrome 同步）
- 定制您的使用体验

### 匿名分析
- 了解扩展的使用方式
- 改进扩展功能和性能
- 发现和修复问题

## 数据共享

我们不会出售、交易或以其他方式将您的信息转让给第三方。

**第三方服务：**
- **PostHog：** 仅限匿名分析数据（可禁用）
- **您的 AI 提供商：** 您选择处理的文本（由您控制）

## 数据安全

- API 密钥存储在浏览器本地的安全存储中
- 与 AI 服务的所有通信均使用 HTTPS
- 我们无法访问您存储的凭据
- 分析数据已匿名化，无法追溯到您

## 您的权利和选择

### 禁用分析
您可以随时禁用匿名分析：
1. 打开扩展设置
2. 关闭"遥测"选项

### 删除数据
- **设置：** 在扩展选项中重置为默认值
- **同步数据：** 通过 Chrome 的同步设置清除
- **分析数据：** 禁用遥测后将停止未来的数据收集

### 导出/导入
您可以随时通过扩展的备份功能导出和导入您的设置。

## 权限说明

| 权限 | 用途 |
|------|------|
| `storage` | 在本地存储您的设置和偏好 |
| `contextMenus` | 为 PDF 文件添加"用 AIction 打开 PDF"选项 |
| `<all_urls>` | 在任意网站上启用文本选择功能 |

## 儿童隐私

本扩展不面向 13 岁以下儿童使用。我们不会故意收集儿童的个人信息。

## 政策变更

我们可能会不时更新本隐私政策。我们将通过以下方式通知您任何变更：
- 更新本政策顶部的"最后更新"日期
- 在扩展文档中发布新政策
